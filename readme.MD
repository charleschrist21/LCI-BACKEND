# Getting Started 

Database :
-change username with your mysql username
-change password with your mysql password
-change database with your mysql database


Run First:
-npm install
-import database.sql to your database
-node App.js

API :

    Admin :
    
        - gets  : /api/admin
        - get   : /api/admin/:id
        - post  : /api/admin
        - put   : /api/admin/:id
        - delete: /api/admin/:id
    Login : 
        - post  : /api/signin
    Jurnal :
        - gets  : /api/jurnal
        - get   : /api/jurnal/:id
        - post  : /api/jurnal
        - put   : /api/jurnal/:id
        - delete: /api/jurnal/:id
